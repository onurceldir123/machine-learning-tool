Returns prediction from POST data. JSON file should be sent compatible with the model.

Example URL:/api/predict/5ef4c5bc35624c9301e4c612

Body parameters: JSON

        
{
    "outlook": "overcast",
    "temperature": "hot",
    "humidity": "high",
    "windy": true
}

Example output

        
{
    "input": {
      "humidity": "high",
      "outlook": "overcast",
      "temperature": "hot",
      "windy": true
    },
    "message": "Prediction was successfully done.",
    "result": "yes"
}


Returns prediction from POST data. JSON file should be sent compatible with the model. It saves to database.

Example URL:/api/predicts/5ef4c5bc35624c9301e4c612

Body parameters: JSON

        
{
    "outlook": "overcast",
    "temperature": "hot",
    "humidity": "high",
    "windy": true
}

Example output

        
{
    "input": {
      "humidity": "high",
      "outlook": "overcast",
      "temperature": "hot",
      "windy": true
    },
    "message": "Prediction was successfully done.",
    "result": "yes"
}
